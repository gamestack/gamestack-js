#### Gamestack.js - a javascript library for 2D games.

##### version 0.8
-------------------------------------------------------------------
#### Features:
 -  javascript game-objects, 2D-graphics
 -  Sprites, Animations, Shots, Motions, Sounds
 -  gamepad-input
 -  events and function-overrides
 -  solutions to common 2D game problems
